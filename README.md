ğŸš€ Getting Started

Prerequisites

Node.js (v18 or higher)

A Google Gemini API Key. You can get one for free at Google AI Studio.

Installation

Clone the repository:

git clone [https://github.com/amanrakesh309/json-prompt-engineer.git](https://github.com/amanrakesh309/json-prompt-engineer.git)
cd json-prompt-engineer


Install dependencies:

npm install


Environment Setup:
Create a .env.local file in the root directory and add your API key:

GEMINI_API_KEY=your_actual_api_key_here


Run the Development Server:

npm run dev


The app will be available at http://localhost:3000.

ğŸ“– Usage Guide

Input: Type your simple idea in the "Your Simple Idea" box.

Refine (Optional): Add an output type hint in the second input field (e.g., "Software Requirement Document").

Select Level: Choose between Basic, Precise, or Advanced based on how much detail you need.

Generate: Press Generate JSON Prompt (or hit Cmd/Ctrl + Enter).

Optimize: Click on the suggested keywords below the output to quickly copy them and add them to your JSON structure.

Deploy: Copy your final JSON and paste it into ChatGPT, Claude, or Gemini as a system instruction or complex prompt.
